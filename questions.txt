What is the difference between an adaptive website and a fully responsive website?
Responsive adapts to the screen size and adaptive is a layout made particular for that screen size.

Describe what it means to be mobile first vs desktop first.
Mobile first means to create the layout for a mobile version first and than add breakpoints for a desktop.

What does font-size: 62.5% in the html tag do for us when using rem units?
It sets the font size to a default size so rem can than multiply off the 62.5.
